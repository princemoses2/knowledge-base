# Knowledge Base

A community-driven collection of helpful programming and development tips.

## Topics Covered
- Git Basics
- GitHub Collaboration
- Command Line Tips
- Git Branching Strategies
- Resolving Merge Conflicts
- Writting Good Commit Messages
## How to Contribute
1. Fork this repository
2. Create a new branch: `topic/your-topic-name`
3. Use TEMPLATE.txt as a guide for creating your topic file
4. Add your file to the `topics/` directory
5. Update this README to include your topic
6. Submit a Pull Request

## File Naming Convention
Use lowercase with hyphens: `your-topic-name.txt`
